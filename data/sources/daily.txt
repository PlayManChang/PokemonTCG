#
# 사용법:
#  1) 아래에 "하고 싶은 말"을 한국어로 한 줄에 하나씩 적어 주세요.
#  2) '#'로 시작하는 줄과 빈 줄은 무시됩니다.
#  3) 다 적으셨으면 Claude에게 "일상용어 반영해줘"라고 말씀해 주세요.
#     → 일본어 + 읽는 법 + 한글 발음으로 번역해서 사이트(용어집 → 일상·여행)에 올립니다.
